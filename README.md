# ⚛️ rlearning

Guía interactiva para aprender React desde cero. Landing page con 14 secciones que cubren todos los conceptos fundamentales de React con demos en vivo y código copiable.

![React](https://img.shields.io/badge/React-18-61dafb?logo=react)
![Tailwind](https://img.shields.io/badge/Tailwind_CSS-v4-06b6d4?logo=tailwindcss)
![Vite](https://img.shields.io/badge/Vite-5-646cff?logo=vite)

## 🔗 Temas cubiertos

JSX · Componentes · Props · useState · useEffect · Renderizado Condicional · Listas y Keys · Formularios · Context API · useRef · Custom Hooks · Memoización · useReducer · Portals

## 🛠️ Tecnologías

- **React 18** — Biblioteca de UI
- **Tailwind CSS v4** — Framework CSS utility-first
- **Vite** — Build tool
- **Lucide React** — Iconos

## 🚀 Ejecutar

```bash
npm install
npm run dev
```
